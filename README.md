Project Overview: Personal Journal Manager
Personal Journal Manager is a lightweight, command-line Python application built to help users quickly capture, review, and manage their daily thoughts. Designed with simplicity and privacy in mind, the app runs entirely in your terminal and saves all data locally without needing an internet connection or external databases.

✨ Key Features
Quick Entry & Auto-Timestamps: Seamlessly add new journal entries, which are automatically tagged with the current date and time.

Chronological Viewing: Read through your complete journaling history in one clean, readable format.

Keyword Search: Easily locate past entries by searching for specific words, phrases, or dates.

Secure Local Storage: All entries are saved directly to a local journal.txt file, ensuring your data remains private and on your machine.

One-Click Reset: Clear your entire journal history with a single, safety-prompted command.

🛠️ Tech Stack & Requirements
Language: Python 3.x

Dependencies: None (Uses only Python's standard built-in libraries like os and datetime).

Interface: Command Line Interface (CLI)
